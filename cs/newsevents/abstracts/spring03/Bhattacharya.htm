An Encryption Scheme for Limited k-time Access to Digital Media

 

Dr. Prabir Bhattacharya

Panasonic Information and Networking Technologies Laboratory

Princeton, NJ

 

Date: Monday, March 10, 2003

Time: 10:00 –11:00 a.m.

(Reception: 11:00 -11:30 a.m.)

Location: Marsh Lounge - Billings

 

Abstract

 

Digital content needs to be protected against piracy yet be readily accessible to the authorized user. Usually, a digital content marketed commercially (such as a software, video etc.) comes with a license that specifies how the authorized user may utilize the content. In this talk we shall describe a new method for protecting digital content from piracy where the "proper use license" is a k-time license that allows the content to be accessed at most k times where k is a pre-determined integer (k >= 1). We provide a scheme that will decrypt and re-encrypt the content at most k times. During each access the complete file is never available in a decrypted form. After accessing the file k times, it gets encrypted with no decryption key available. Since secret keys must reside on the user's system, we heighten the level of security by hiding the secret keys in the heuristic solutions of NP-hard problems. The method provides a renewable security mechanism because any NP-hard problem and any approximate solution could be used and the secret keys can be changed at will from time to time. This is a work in collaboration with Gregory Perkins.
