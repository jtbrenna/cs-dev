Abstract

 

Ontologies and conceptual models are useful for semantic annotation of data and for providing an integrated view of heterogeneous data sources. The increasing usage of ontologies in semantically driven data integration makes the idea of an ontology-based user interface desirable. Recognizing the necessity of providing intuitive graphical metaphors to simplify the interaction between the user and the ontology formalism, we have developed GrOWL, a graphic framework for visualization, browsing and editing of OWL ontologies. This presentation    describes GrOWL-Query -the visual query language based on GrOWL.   The discussion will be focused on   formal semantics and analysis of expressive power of GrOWL-Query.
