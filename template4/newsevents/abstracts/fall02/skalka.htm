Abstract

 

We present pml, a programming language that includes primitive sets and associated operations. The language is equipped with a precise type discipline that statically captures dynamic properties of sets, allowing runtime optimizations. We demonstrate the utility of pml by showing how it can serve as a practical implementation language for higher-level systems: it can express several forms of programming language-based security, and implement monomorphic flow types.
