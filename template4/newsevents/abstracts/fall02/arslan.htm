Dictionary Look-Up within Small Edit Distance

 

Dr. Abdullah Arslan

Assistant Professor

Department of Computer Science

The University of Vermont

 

Date: Wednesday, November 13, 2002

Time: 3:30 p.m. – 4:30 p.m.

Location: 105 Votey

 

Abstract

Let W be a dictionary consisting of n binary strings of length m each, represented as a trie.  The usual d-query asks if there exists a string in W within Hamming distance d of a given binary query string q. We present a simple algorithm to determine if there is a member in W within edit distance d of a given query string q of length m. The method takes time O(dmd+1) in the RAM model, independent of n, and requires O(dm) additional space. We also generalize the results for the case of the problem over a larger alphabet.
